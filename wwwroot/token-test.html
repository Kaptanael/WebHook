Token-Based Webhook Tester

POST /api/webhook with X-Endpoint-Token (base64url-encoded), X-Signature (HMAC-SHA256), and X-Timestamp headers. Simulates submitting events to subscribed endpoints.

1. Token Setup

Create or paste a base64url-encoded token containing decoded fields: BaseUrl, ApiKey, ClientId, ClientSecret, AppName, CompanyId (newline-separated).

Format: base64url(BaseUrl\nApiKey\nClientId\nClientSecret\nAppName\nCompanyId)
