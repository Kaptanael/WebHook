x-token Inbound Tester

POST /api/inbound with only the x-token header — no timestamp, signature, or api-key. The event type comes from the body's top-level type (Standard Webhooks envelope). Other testers: standard-webhooks · token · all-in-one

Request

The raw endpoint token; matched against the endpoint's EndPointToken. This is the only credential header sent.
Must be non-empty and carry a top-level type (the event type). Delivered to the endpoint verbatim.
